# Product

## Register

product

## Users
Nihat — founder/solo operator at AI-Next. Uses the CRM dashboard daily for morning outreach reviews. Primary context: desktop browser, morning routine, needs to scan pipeline health and take next actions without opening each record.

## Product Purpose
A standalone HTML CRM dashboard tracking a B2B cold outreach pipeline (15 Ankara companies). Syncs bidirectionally with Google Sheet via Apps Script. Automated agents run at 09:09, 09:23, 09:57, and 22:00 on weekdays; the dashboard surfaces what the agents did and what needs a human touch today.

## Brand Personality
Minimal, operational, sharp. Matches the AI-Next main site: monochromatic, editorial, zero ornamentation.

## Anti-references
- Colorful SaaS dashboards (HubSpot/Salesforce density + color saturation) — too noisy
- Shadow-heavy card grids — off-brand
- Pastel pill-heavy Notion-style UIs — too soft for a sharp daily tool

## Design Principles
1. Urgency visible at a glance — overdue items must be obvious without clicking
2. One action per click — stage and LinkedIn updates happen inline from the table
3. Data is the design — chrome disappears; pipeline data is the interface
4. Match the AI-Next brand — zero border-radius, thin 1px borders, #C22C2F accent, Inter only
5. Dashboard first — the agent schedule and pipeline health show on load, not buried in menus

## Accessibility & Inclusion
WCAG AA minimum. Keyboard navigation (Escape closes panel). No specific user needs beyond standard.
